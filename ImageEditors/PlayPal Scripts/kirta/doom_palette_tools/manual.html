Doom Palette Tools

Tools for converting between SLADE Doom palettes and strip formats. Designed for Doom / MBF21 / DSDA workflows.

Included tools

  • SLADE → Strip (1x1)
    Converts a 16×16 SLADE palette PNG into a 256×1 palette strip.
  • SLADE → Strip (8×8)
    Converts a 16×16 SLADE palette PNG into a 2048×8 strip (256 colours as 8×8 blocks).
  • Strip → SLADE (1x1)
    Converts a 256×1 palette strip back into a SLADE-style palette PNG.
  • Strip → SLADE (8×8)
    Converts a 2048×8 strip back into a 128×128 SLADE palette PNG.

Notes

  • All exports are silent (no dialogs)
  • PNG compression = 0
  • Indexed PNG when possible
  • Files are saved next to the source image
