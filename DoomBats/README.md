# DoomBats

G'Day aJynks here....

This is just a easy to install version of my doomScripts. Simply copy the dir into your path and add it, as well as any sub directories to your path.

You're welcome to use these in anyway you wish, they are designed for me, but if you want to use them.. go for it!

This includes...
- **doommake-tweak** : Utility that edits a fresh doomtools project dir to my personal prefrences. Designed for MBF21 wads
- **doomRun**    	 : Simple Doom Launcher
- **doomCube**   	 : Build a LUT Cube for realtime palette conversions ajustment layers, in sRGB documents
- **drawMaps**   	 : Export Maps from a doom wad into PNGs
- **playpal**	 	 : Utility to edit and convert doom playPals
- **dmake**			 : wrapper for doommake. 
- **mdoom**		 	 : Small utility to add a config dir to a doom wad dir for screens, demos and savegames (managment tool for use with DoomRunner
- **updatebats** 	 : Small utility to download bats to any system I happen to be working on
